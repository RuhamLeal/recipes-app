## Getting Started



### Prerequisites

node 16 version

### Installation

```
Clone the repo:   
$ git clone git@github.com:RuhamLeal/ExpressJS-BookStore-API.git    

Go to project folder:     
$ cd ExpressJS-BookStore-API     

Install dependencies:    
$ npm install

Run app:   
$ npm start

```
